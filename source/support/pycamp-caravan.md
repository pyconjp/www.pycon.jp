# Python Boot Camp Caravan

Python Boot camp Caravan（略称PyCamp Caravan）は、[Python Boot Camp](https://www.pycon.jp/support/bootcamp.html) や [PyLadies Caravan](https://tokyo.pyladies.com/caravan/index.html) 等の事業や毎年行われる PyCon JP等のイベントの周知を図るため、全国各地で開催される [オープンソースカンファレンス](https://www.ospn.jp/) などのイベントに出展をしています。

ブース展示は Python Boot Camp / PyLadies Caravan の開催に興味のある方と出会うことを目的としており、イベントの趣旨や開催要件などをお伝えしています。<br>
また、ブースに来てくれた方へステッカー配布や Python に関するアンケート等を実施しています。

セミナーは参加するメンバーが選んだトークテーマで Python に関する講演を行っています。
